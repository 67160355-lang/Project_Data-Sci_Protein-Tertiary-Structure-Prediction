วัตถุประสงค์ (Objectives)
เพื่อพัฒนาแบบจำลอง Machine Learning แบบ Binary Classification สำหรับจำแนกคุณภาพของโครงสร้างโปรตีนออกเป็น 2 กลุ่ม ตามระดับความคลาดเคลื่อนทางโครงสร้าง (RMSD) เพื่อช่วยให้นักวิจัยคัดกรองโครงสร้างที่มีความแม่นยำสูงได้อย่างรวดเร็ว

เพื่อวิเคราะห์ปัจจัยหลัก (Key Features) ที่มีอิทธิพลต่อความเสถียรและความถูกต้องของโครงสร้างโปรตีน เช่น สัดส่วนพื้นที่ผิวไม่ชอบน้ำ (Hydrophobicity) และน้ำหนักโมเลกุล (MolecularWeight) ผ่านเทคนิค Feature Importance

เพื่อสร้างเครื่องมือช่วยตัดสินใจ (Decision Support Tool) ในรูปแบบ Web Application ที่แสดงค่าความเชื่อมั่น (Probability Score) ช่วยลดระยะเวลาและงบประมาณในกระบวนการคัดกรองโครงสร้างโปรตีนเบื้องต้นในงานด้านการออกแบบยา (Drug Discovery)

2. การนิยามกลุ่มเป้าหมาย (Target Classes)
การเปลี่ยนจาก Regression เป็น Classification ช่วยลดผลกระทบจากข้อมูลที่ผิดปกติ (Robustness against Outliers) โดยแบ่งกลุ่มตามเกณฑ์ความคลาดเคลื่อนทางโครงสร้าง (RMSD) ดังนี้:

Class 0 (High Quality): ค่า RMSD<5 Å หมายถึงโครงสร้างมีความใกล้เคียงกับสถานะธรรมชาติ (Native State) สูง มีความน่าเชื่อถือในการนำไปศึกษาต่อ

Class 1 (Low Quality): ค่า RMSD≥5 Å หมายถึงโครงสร้างมีความคลาดเคลื่อนสูง หรือมีการม้วนตัวที่ผิดเพี้ยน (Mis-folded)

3. คุณลักษณะที่ใช้ในการทำนาย (Selected Features)
คัดเลือก 4 ปัจจัยทางกายภาพและเคมีที่มีความสัมพันธ์โดยตรงกับเสถียรภาพของโปรตีน:

รหัส	คุณลักษณะ (Feature)	ความสำคัญทางชีวภาพ
F3	Hydrophobic Surface Ratio	สัดส่วนพื้นที่ผิวไม่ชอบน้ำ ยิ่งค่าต่ำในส่วนที่สัมผัสภายนอก ยิ่งสะท้อนการม้วนตัวที่สมบูรณ์
F4	Polar Exposed Area	พื้นที่ผิวส่วนชอบน้ำที่สัมผัสสิ่งแวดล้อม ช่วยรักษาความเสถียรผ่านพันธะไฮโดรเจน
F2	Non-polar Exposed Area	พื้นที่ผิวส่วนไม่ชอบน้ำที่หลุดรอดออกมาภายนอก (ถ้าสูงเกินไปอาจหมายถึงโครงสร้างพัง)
F9	Molecular Weight	น้ำหนักโมเลกุลรวม ใช้เป็นตัวควบคุมขนาด (Scale) ของพื้นที่ผิวทั้งหมด
4. วิธีการดำเนินงาน (Methodology)
4.1 อัลกอริทึม (Algorithm)
เลือกใช้ Random Forest Classifier เนื่องจาก:

สามารถจัดการกับความสัมพันธ์ที่เป็น Non-linear ระหว่าง Features ได้ดี

ให้ค่า Probability Score (ความน่าจะเป็น 0-100%) ทำให้ผู้ใช้งานทราบว่าโมเดล "มั่นใจ" แค่ไหนในการจำแนก

มีความทนทานต่อข้อมูลที่มีความแปรปรวนสูงในทางชีวสารสนเทศ

4.2 การวัดผลโมเดล (Evaluation Metrics)
Accuracy: ความแม่นยำรวมในการจำแนกกลุ่ม

F1-Score: ค่าเฉลี่ยที่สมดุลระหว่าง Precision และ Recall เพื่อยืนยันว่าโมเดลไม่เอนเอียงไปที่กลุ่มใดกลุ่มหนึ่ง

Confusion Matrix: เพื่อดูอัตราการทำนายผิดประเภท (เช่น การทำนายโปรตีนคุณภาพต่ำว่าเป็นคุณภาพดี ซึ่งต้องระวังเป็นพิเศษ)
